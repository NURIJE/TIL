# Comment (주석)


한줄 주석
~~~javascript
// 책을 book 변수에 할당합니다.
var book = "책"; // 여기부터 주석
~~~
<br>

블록 주석
~~~javascript
/*
var book = "책";
var music = "음악";
*/
~~~
<br>

/** 프로그램 설명 */
~~~javascript
/**
* @function getName
* 1. 이 형태로 작성
* @param {String} code, 코드
*/
function getName(code){};
~~~
- 개발자들 사이의 코딩 관례
- 이렇게 작성하면 프로그램 설명문서를 자동으로 만들어 주는 툴이 있음.
