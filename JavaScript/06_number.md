# Number
- 정수, 실수, 숫차 처리

<br>


## JS는 정수, 실수를 구분하지 않음
  - 1, 1., 1.0 모두 1.0으로 간주
  - 단, 표시는 1로 표시

<br>

~~~javascript
console.log(0.12 + 5);
// Output : 5.12
~~~
- 정수와 실수를 구분하는 언어에서는 실수(0.12)와 정수(5)를 더할 수 없음
- 정수를 실수(5.0)를 변환해야 가능
- JS는 정수와 실수를 구분하지 않고 실수로 계산함.
  
<br>

## IEEE 754 표준 준수
  - 64Bit 부동 소수점 처리
  - 1도 64bit, 1억도 64bit로 처리

## ES6에 정수, 실수 구분 추가
- 정수, 실수 구분이 없는 것은 장단점이 있음
- ES6에서는 단점을 극복하기 위해 정수, 실수 구분을 추가함.
